# PartyAR

Party AR is a web app that use Augmented Reality to play 3D scenes and animate them using Audio frequencies.

## Require:
* Node.js > 20
* React > 18
* Vite > 4
* Swc 
* react-tensorflow
* @react-three/fiber
* @react-three/drei
* lerp
* zustand
